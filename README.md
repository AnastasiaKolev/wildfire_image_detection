# Wildfire Image Detection using Pretrained Vision Transformer

## Описание проекта
В рамках проектного практикума разработан прототип IT-решения для мониторинга экосистем,
направленный на автоматическое выявление лесных пожаров по изображениям.

Решение использует предобученную модель компьютерного зрения (Vision Transformer, ViT),
которая классифицирует изображения на два класса:
- fire — присутствует пожар
- no_fire — пожара нет

Цель проекта — продемонстрировать, как готовые модели машинного обучения
могут применяться для экологического мониторинга и поддержки принятия решений.

---

## Бизнес-постановка задачи
Лесные пожары представляют серьёзную угрозу для экосистем, животных и людей.
Раннее обнаружение пожара позволяет:
- быстрее реагировать на чрезвычайные ситуации;
- снизить экологический и экономический ущерб;
- автоматизировать мониторинг больших территорий.

Задача проекта — определить, присутствует ли пожар на изображении,
полученном, например, с дронов, камер наблюдения или спутников.

---

## Постановка задачи в терминах Data Science
- Область: Computer Vision (CV)
- Тип задачи: бинарная классификация изображений
- Входные данные: изображения лесных территорий
- Выходные данные: класс изображения (fire / no_fire) и вероятность предсказания

---

## Используемая модель
В проекте используется предобученная модель Vision Transformer (ViT).

Источник примера и архитектуры:
https://www.kaggle.com/code/dima806/wildfire-image-detection-vit

Причины выбора модели:
- ViT показывает высокое качество на задачах классификации изображений;
- модель уже обучена на больших наборах данных;
- не требует дообучения для демонстрации прототипа;
- легко интегрируется в Python-пайплайн.

---

## Данные
Для демонстрации работы модели используются изображения лесных территорий,
содержащие сцены с пожарами и без них.

Источник данных указан в ноутбуке (Kaggle).

## Интерпретация результатов

Модель возвращает:

* предсказанный класс (fire / no_fire); 
* вероятность предсказания.

В рамках системы мониторинга экосистем рекомендуется:

* использовать автоматическое оповещение при высокой вероятности пожара; 
* при вероятности ниже заданного порога (например, 70%) 
направлять изображения на ручную проверку специалистом.

## Ограничения решения

* Модель может ошибаться при плохом качестве изображения; 
* Возможны ложные срабатывания при наличии дыма, тумана или закатного освещения; 
* Решение является прототипом и не предназначено для промышленного использования.

## Выводы

В ходе проекта продемонстрировано, как предобученные модели машинного обучения
могут быть использованы для экологического мониторинга.
Даже без дообучения модель позволяет выявлять пожары на изображениях
и может служить основой для дальнейшего развития системы.
